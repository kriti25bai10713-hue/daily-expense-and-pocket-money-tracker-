# Problem Statement
Managing daily expenses and pocket money manually is prone to errors and lack of clarity. People often face difficulty tracking income and expenses effectively, leading to poor budgeting and financial management. The goal is to develop an easy-to-use digital tracker that allows users to record, view, and analyze their daily transactions efficiently.

# Scope of the Project
This project covers daily income and expense tracking functionalities that record transactions with details such as date, amount, type (income or expense), category, and notes. It provides clear menus for adding transactions, viewing all transactions, and generating summary reports including total income, expenses, and balance. The project stores data persistently in a simple text file, making it lightweight and easy to maintain.

# Target Users
Students receiving and managing pocket money.

Individuals wanting to track small-scale daily finances without complexity.

Beginners looking for a simple and interactive personal finance tool.

# High-level Features
Add income or expense entries with date, amount, category, type, and notes.

View all recorded transactions in a readable, chronological format.

Generate summary reports showing total income, total expenses, and balance.

Simple text file-based data storage for persistent and portable data management.

Interactive menu-driven interface enabling straightforward user interaction.

Modular and clean Python codebase facilitating easy understanding and extension.
